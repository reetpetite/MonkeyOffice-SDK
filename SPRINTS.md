# Entwicklungszyklen

## Sprint 4 – Research Lab

### Ziel

Experimente reproduzierbar definieren, ausführen, importieren und über Builds vergleichen.

### Enthalten

- Experimentdefinitionen in YAML
- Testskriptgenerator
- Parser für kopierte `msgBox`-Ausgaben
- Speicherung der Rohdaten
- Forschungsberichte
- Buildvergleich
- Forschungsvalidator
- Referenzdaten für MO-027 und MO-028
- vorbereitete Testserie MO-029

### Definition of Done

- [x] Experimente besitzen eine eindeutige Verzeichnisstruktur
- [x] MonKey-Skripte werden aus YAML erzeugt
- [x] alle Ausdrücke bleiben in einer einzigen `msgBox`-Scriptzeile
- [x] Ergebnisse können aus kopiertem Text importiert werden
- [x] Rohdaten bleiben unverändert erhalten
- [x] unterschiedliche Builds können verglichen werden
- [x] vorhandene Ergebnisse MO-027 und MO-028 sind integriert
- [x] MO-029 ist ausführungsbereit
