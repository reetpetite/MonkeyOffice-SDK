# MonkeyOffice SDK

Eine experimentell geprüfte Dokumentation der Skriptsprache für
Bankimportdefinitionen in MonKey Office.

## Ziele

- präzise Sprachreferenz
- reproduzierbare Funktionstests
- dokumentierte Randfälle und bekannte Fehler
- Importdefinitionen für verschiedene Banken und Zahlungsdienste
- Best Practices für robuste Importskripte

## Status

Dieses Projekt basiert auf:

- offizieller Dokumentation
- reproduzierbaren Experimenten
- realen Importdefinitionen

Erkenntnisse werden gekennzeichnet als:

- 📖 offiziell dokumentiert
- 🧪 experimentell verifiziert
- 💡 Empfehlung
- ⚠️ bekannte Gefahr oder Fehler

## Unterstützte Definitionen

- N26
- comdirect
- PayPal

## Haftungsausschluss

Dieses Projekt ist unabhängig vom Hersteller von MonKey Office.
Importdefinitionen sollten zunächst immer mit Testdaten geprüft werden.
